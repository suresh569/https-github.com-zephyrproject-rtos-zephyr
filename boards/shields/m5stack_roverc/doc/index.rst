M5Stack roverc
##############

Overview
********
The M5Stack Roverc is a programmable Mecanum wheel omnidirectional mobile robot base.
Compatible with M5StickC/M5StickC PLUS.

Requirements
************

This shield can only be used with the `m5stick_cplus` board.
